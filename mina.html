Publish mina.ai for free on GitHub Pages

Beginner‑friendly steps. No paid hosting needed.

Option A — Fastest (single HTML file)

  1. Create a GitHub repo (public) e.g. mina-ai
  2. Add this file as index.html in the repo
  3. Repo → Settings → Pages
  4. Source: Deploy from a branch
  5. Branch: main and folder /root
  6. Open the provided URL (GitHub shows it on the Pages screen)

Updating prompts (later)

  1. Open index.html on GitHub
  2. Click ✏️ Edit
  3. Find const prompts
  4. Add a new prompt object (copy an existing one)
  5. Commit changes → site updates automatically

Pro tip (custom domain later)

  1. Buy mina.ai domain (any registrar)
  2. In GitHub Pages: set Custom domain
  3. Add DNS records as GitHub instructs
